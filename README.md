## Prequesites

- You need the latest Node 20 LTS version installed [Node.js](http://nodejs.org)

## Install

First install all dependencies:

```bash
$ npm install
```

## Usage

After you've installed the dependencies you can run the application:

```bash
$ npm run start
```
